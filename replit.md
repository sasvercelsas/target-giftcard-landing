# Target Gift Card Landing Page

## Overview
A static landing page built with pure HTML and CSS (no JavaScript/TypeScript). Features a light color scheme with Target branding, a prominent CTA button linking to an affiliate URL, and a "How It Works" section.

## Project Architecture
- `index.html` - Main landing page
- `styles.css` - All styling
- `serve.py` - Simple Python HTTP server for serving static files on port 5000

## Key Details
- No JavaScript or TypeScript used anywhere
- Button links to: https://trkfy.org/aff_c?offer_id=250&aff_id=164586
- Light color scheme with red Target brand accents
- Responsive design for mobile and desktop
